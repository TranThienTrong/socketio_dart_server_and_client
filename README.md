# Getting Started
**Step 1:** Run dart_server.dart\
**Step 2:** Android Emulator has problem with port 3000, so remember to use **adb reverse tcp:3000 tcp:3000**\
**Step 3:** Run dart_client.dart


To test for other requests, go to:\
